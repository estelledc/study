# Llama 2 学习笔记

## 一、这篇论文在说什么（一句话）

Meta 发布了 Llama 2 —— 一套 **7B / 13B / 70B** 三种规模的开源大模型，并在 Llama 1 基础上做了全面升级，特别推出了专门针对对话场景微调的 **Llama 2-Chat**。

## 二、日常类比

想象你在教一个学生：

1. **预训练（Pre-training）** = 让学生读完整个图书馆的书。Llama 2 读了 **2 万亿 token**（比 Llama 1 的 1 万亿多了一倍），而且用了更大的上下文窗口（4096 vs 2048）。
2. **监督微调（SFT）** = 老师一对一辅导，告诉学生"这样回答问题更好"。
3. **RLHF（人类反馈强化学习）** = 老板给员工打分，得分高的做法被鼓励。Llama 2 在这一步做了重大改进。

## 三、核心概念

### 3.1 架构：Transformer Decoder-only

Llama 2 使用和 Llama 1 相同的架构，但有几个关键升级：

- **SwiGLU 激活函数**：把原来的 GeLU 换成了 SwiGLU（Swish-Gated Linear Unit），效果更好。
- **RMSNorm**：用更简单的归一化方法替代 LayerNorm。
- **位置编码 RoPE**：旋转位置编码，让模型能处理更长的上下文。

### 3.2 训练三阶段

| 阶段 | 做什么 | 类比 |
|------|--------|------|
| 预训练 | 从海量文本中学习语言规律 | 读完整个图书馆 |
| SFT 微调 | 用高质量指令数据微调 | 老师一对一辅导 |
| RLHF | 用人类偏好数据做强化学习 | 老板打分激励 |

### 3.3 安全改进

这是 Llama 2 最大的亮点之一。论文中专门用一个章节讲安全：

- 收集了 **9,200+ 标注数据**用于 SFT 训练
- 用 **480 万条数据**做 RLHF 偏好训练
- 引入了 **红队测试（Red Teaming）**，主动攻击自己的模型来发现漏洞

## 四、代码示例

### 示例 1：用 transformers 库加载 Llama 2

```python
from transformers import AutoTokenizer, AutoModelForCausalLM

# 加载 7B 版本的 Llama 2
model_name = "meta-llama/Llama-2-7b-chat-hf"

tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForCausalLM.from_pretrained(
    model_name,
    torch_dtype="auto",
    device_map="auto"
)

# 构造对话输入
messages = [
    {"role": "system", "content": "你是一个有帮助的助手。"},
    {"role": "user", "content": "什么是机器学习？请用简单的话解释。"}
]

input_text = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
inputs = tokenizer(input_text, return_tensors="pt").to(model.device)

# 生成回复
outputs = model.generate(**inputs, max_new_tokens=256)
response = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(response)
```

**关键点**：
- `apply_chat_template` 自动组装对话格式，不需要手动拼 prompt
- `device_map="auto"` 自动分配到 GPU/CPU

### 示例 2：用 LoRA 做轻量级微调

LoRA（Low-Rank Adaptation）是一种高效的微调方法，只训练少量参数而不改动原模型。

```python
from peft import LoraConfig, get_peft_model
from transformers import AutoModelForCausalLM, TrainingArguments, Trainer
from datasets import load_dataset

# 加载基座模型
base_model = "meta-llama/Llama-2-7b-hf"
model = AutoModelForCausalLM.from_pretrained(base_model, torch_dtype="auto")

# 配置 LoRA
lora_config = LoraConfig(
    r=8,                    # 低秩矩阵的秩（越小越省显存）
    lora_alpha=32,          # 缩放系数
    target_modules=["q_proj", "v_proj"],  # 只对 q 和 v 矩阵做 LoRA
    lora_dropout=0.05,
    bias="none",
    task_type="CAUSAL_LM"
)

# 包装成可训练的 LoRA 模型
model = get_peft_model(model, lora_config)
model.print_trainable_parameters()
# 输出类似: trainable params: 4,194,304 || all params: 6,743,763,200 || 0.06%

# 准备数据集
dataset = load_dataset("tatsu-lab/alpaca")
train_dataset = dataset["train"]

# 训练设置
training_args = TrainingArguments(
    output_dir="./llama2-lora-finetune",
    num_train_epochs=3,
    per_device_train_batch_size=4,
    learning_rate=2e-4,
    fp16=True,
    logging_steps=10,
)

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=train_dataset,
)

trainer.train()
model.save_pretrained("./llama2-lora-finetuned")
```

**关键点**：
- 原始模型 67 亿参数，LoRA 只训练 420 万参数（约 0.06%）
- `target_modules` 指定哪些层加 LoRA 适配器，通常是注意力层的 q_proj 和 v_proj
- 微调后只需保存这个小适配器，推理时自动合并到原模型

## 六、我的理解

Llama 2 最重要的贡献不是某个单一技术突破，而是把"开源 + 高质量 + 安全"这三件事同时做到了。之前开源模型的对话能力远不如 GPT-4 等闭源模型，但 Llama 2-Chat 在人类评估中已经可以"作为闭源模型的合适替代品"——这句话的分量很重。
